# MioLica
Miolica is an e-commerce application that uses React.js as the frontend and Laravel as the backend.

## MioLica Tech Stack
1. Frontend
    - Typescript
    - React.js
    - Redux.js
    - Tailwind CSS
    - Progressive Web Apps
2. Backend
    - PHP
    - Laravel (Only API)
    - Postgre SQL
    - GraphQL
    - Sanctum (API token)
