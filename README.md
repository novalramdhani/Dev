## MioLica Tech Stack
1. Frontend
    - Typescript
    - React.js
    - Redux.js
    - Tailwind CSS
    - Progressive Web Apps
2. Backend
    - PHP
    - Laravel (Only API)
    - PostgreSQL
    - MongoDB
    - GraphQL
    - Sanctum (API token)
